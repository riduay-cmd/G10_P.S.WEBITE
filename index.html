Platform Security and Cybersecurity

Platform security involves a centralized approach to protect the entire IT infrastructure, including its components and layers, by integrating and managing security controls and policies across the entire "platform".

Cyber security is the broader field encompassing technologies, processes, and controls used to defend computer systems, networks, and data from digital attacks and unauthorized access.

Virtualization

Virtualization

A powerful technology that enables multiple virtual machines (VMs) to run concurrently on a single physical server.

TYPES

SERVER-Multiple virtual servers on one physical server.Most common type.

DESKTOP-Virtual desktop infrastructure (VDI).Remote desktop access.

NETWORK-Virtual networks independent of physical hardware.

STORAGE-Pool storage from multiple devices.

APPLICTION-Applications run in isolated environments.

BENEFITS

Cost Reduction

Increased Agility

Enhanced Disaster Recovery

Improved Management

Testing & Compatibility

POPULAR PLATFORMS

Microsoft Hyper-V

Citrix XenServer

Red Hat Virtualization

Trusted Platform Module (TPM)

TPM is a hardware-based security chip Provides a Root of Trust for a computing platform Functions independently of the main CPU and OS Ensures secure storage, encryption, and attestation.

VERSIONS

TPM 1.2: Basic cryptographic support (SHA-1, RSA).

TPM 2.0: Stronger crypto algorithms (SHA-256, ECC), flexible authorization.

BENEFITS

Protects against:Rootkits & bootkits,Key theft,Unauthorized firmware changes.

Provides hardware-enforced trust instead of relying on software-only security.

Unified Extensible Firmware Interface (UEFI)

It’s basically the modern replacement for BIOS (Basic Input/Output System), which is the traditional firmware that starts your computer when you power it on.

Hypervisor

A software layer that manages and allocates physical hardware resources to virtual machines, ensuring their isolation and efficient operation.

TYPES

Type 1: Bare-Metal- Runs directly on hardware, offering high performance and security (e.g., VMware ESXi, Microsoft Hyper-V).

Type 2: Hosted Runs- on top of a host operating system (e.g., VirtualBox, VMware Workstation).
